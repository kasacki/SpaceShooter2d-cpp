Detailed Description

Klasa reprezentująca trzeci typ obcego (Alien C).

  • Dziedziczy po klasie Alien i implementuje jej czysto wirtualne metody. Obsługuje ładowanie i rozładowywanie własnych tekstur oraz specyficzne zachowanie.
