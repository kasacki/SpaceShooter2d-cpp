Detailed Description

Klasa zarządzająca logiką i stanem gry.

Klasa Game obsługuje inicjalizację, aktualizację i rysowanie wszystkich elementów gry, takich jak statek kosmiczny, obcy, pociski, przeszkody i statek-zagadka. Zarządza również punktacją, życiem gracza, logiką menu oraz zapisem/odczytem najlepszego wyniku.
