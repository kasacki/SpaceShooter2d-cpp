\doxysubsection{Detailed Description}
Klasa bazowa dla wszystkich typów obcych w grze. 


\begin{DoxyItemize}
\item Klasa \doxyref{Alien}{p.}{class_alien} definiuje wspólny interfejs i podstawowe właściwości dla różnych typów obcych. Jest to klasa abstrakcyjna, co oznacza, że nie można tworzyć jej bezpośrednich instancji, a klasy pochodne muszą implementować jej czysto wirtualne metody. 
\end{DoxyItemize}
