\doxysubsection{Detailed Description}
Klasa zarządzająca logiką i stanem gry. 

Klasa \doxyref{Game}{p.}{class_game} obsługuje inicjalizację, aktualizację i rysowanie wszystkich elementów gry, takich jak statek kosmiczny, obcy, pociski, przeszkody i statek-\/zagadka. Zarządza również punktacją, życiem gracza, logiką menu oraz zapisem/odczytem najlepszego wyniku.
